# Image-Rich Recommendation Sections

**Date:** 2026-04-15
**Goal:** Replace text-heavy home page recs with image-rich, category-specific horizontal scroll strips with real cover art and direct CTAs to Netflix, Spotify, Amazon, Zomato etc.

## Sections

1. **Watch Tonight** — movie/show posters from TMDB API
2. **Listen Now** — album art from existing Apple Music/iTunes data
3. **Read Next** — book covers from Google Books API
4. **Eat Here** — restaurant cards from web search
5. **Explore** — event cards from web search

## Data Sources

| Category | Image Source | Metadata Source | CTA |
|----------|------------|-----------------|-----|
| Movies | TMDB poster API | Claude recs from identity | netflix.com/search, primevideo.com/search |
| Music | artworkMap (iTunes) | Apple Music/Spotify identity | open.spotify.com/search, music.apple.com |
| Books | Google Books thumbnails | Claude recs from identity | amazon.in/s?k= |
| Restaurants | Web search thumbnails | Claude + web search | zomato.com/search |
| Events | Web search thumbnails | Claude + web search | Direct URLs |

## API

**`POST /api/home/rich-recs`**
- Input: `{ googleSub, profile (interests, location, music, etc) }`
- Claude generates 3-4 recs per category based on identity
- For movies: fetch TMDB for poster + metadata
- For books: fetch Google Books for cover
- For music: use existing artworkMap
- For restaurants/events: use existing web search
- Cached in content cache (24hr TTL)
- Returns: `{ movies: [...], music: [...], books: [...], restaurants: [...], events: [...] }`

Each item: `{ title, subtitle, image, tag, matchReason, ctaLabel, ctaUrl }`

## New Env Var

`TMDB_API_KEY` — free from themoviedb.org

## Component

**`RecommendationStrip.svelte`** — generic horizontal scroll strip
- Props: `title, emoji, items[], variant: 'tall' | 'square' | 'wide'`
- tall = movie posters (2:3 aspect)
- square = album art
- wide = restaurants/events (16:9)
- Each card: image, title, subtitle, tag pill, match reason, CTA button
- Glass card style, stagger animation, hover lift
